Optimization Algorithms in Python

FAU · 2024 — implementing numerical optimization methods in Python.

What I implemented

  • Wrote clean Python scripts for unconstrained optimization with reusable functions and tests.
  • Added convergence diagnostics (grad-norm, step length, iteration caps) and timing/iteration tracking.
  • Integrated line-search (backtracking/Wolfe) and a basic trust-region variant for robustness.

Focus on

  • Optimality conditions for smooth problems and basics of KKT.
  • Descent methods with exact/inexact line search and convergence rates.
  • Conjugate-gradient and Newton-type methods (incl. quasi-Newton).
  • Trust-region strategies and Levenberg–Marquardt for least-squares.

Algorithms coded

  • Gradient Descent with backtracking / Wolfe line search.
  • Nonlinear Conjugate Gradient (Fletcher–Reeves / Polak–Ribiere).
  • BFGS (and L-BFGS style update) with inexact line search.
  • Newton & Levenberg–Marquardt for least-squares problems.
  • Trust-Region (Cauchy / CG-Steihaug step) for globally convergent behavior.
