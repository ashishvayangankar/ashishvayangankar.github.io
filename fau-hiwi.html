AbqHomo: Python–Abaqus Automation for RVE Homogenization

Jun 2023 – Sep 2024 · Erlangen, Germany

Task

I developed AbqHomo, a utility that automates homogenization of composite RVEs in Abaqus using periodic boundary conditions (PBCs), and computes macro-stress and the full macro-tangent (effective property matrix) for 2D/3D—including piezoelectric coupling.

From microstructure to effective properties

To model large components made of heterogeneous materials, we need a single “effective” material. AbqHomo takes a small periodic micro-model (RVE) like the one shown and applies PBCs (Periodic Boundary Conditions), solves a set of unit loadings in Abaqus, and averages the response to obtain macro-level properties.

  • Works with orphan meshes; automatic periodic pairing + constraint equations with helper (dummy) nodes.
  • GUI/noGUI operation; suitable for batch studies and cluster runs.
  • Extends to electro-mechanical coupling (piezoelectric RVEs).

What AbqHomo computes

For each unit loading, AbqHomo returns the volume-averaged macro-stress. By running a complete set of unit cases, it assembles the effective-property (or macro-tangent) matrix. For piezoelectric RVEs, the matrix is block-structured: mechanical stiffness C*, piezoelectric coupling e*, and dielectric permittivity ε* (Voigt notation).

How it works (overview)

  • Ingestion & setup: I designed AbqHomo to read existing Abaqus meshes and a compact config describing the RVE bounds and physics (mechanical or piezoelectric). The tool then builds the analysis model programmatically.
  • Robust periodicity: it automatically identifies opposite faces, pairs nodes, and applies periodic boundary conditions with guard checks (incl. dummy nodes for non-matching/orphan meshes), eliminating manual GUI work.
  • Automated solves & averaging: AbqHomo orchestrates the set of unit load cases in Abaqus and performs volume-averaging to compute the macro-stress response of the microstructure.
  • Effective properties: from those runs the tool assembles the full macro-tangent—mechanical stiffness C*, piezoelectric coupling e*, and permittivity ε*—ready to use as homogenized material data in component-scale FE models.
  • Outputs & scale: results are written as clean tables/plots for direct reporting or import; the workflow is batchable for parameter studies and cluster execution.
