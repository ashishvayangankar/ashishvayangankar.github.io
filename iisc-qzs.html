Quasi-Zero Stiffness (QZS) Vibration Isolator

Jul 2021 – Sep 2022 · Bangalore, India

What we set out to build

A quasi-zero stiffness (QZS) isolator combines a positive-stiffness element (linear spring) with a negative-stiffness element so that, over a design window, the force–displacement curve is nearly flat. That “flat zone” lets the base move while the payload sees minimal transmitted force — ideal for isolating low-frequency vibration.

Design and static characterization

  • Built the QZS cell using a linear spring and a pre-buckled/geometry-based negative-stiffness element.
  • Ran displacement-controlled tests to map the force–displacement curve and identify the QZS window.
  • 3D-printed TPU versions showed hysteresis; switching to spring steel yielded a clean, repeatable response.

Dynamic tests and transmissibility

With the static window set, we ran sine tests across frequency and amplitude on a base-excitation rig and evaluated transmissibility (output/input). The isolator reliably reduced transmitted motion at low frequencies (≈ up to 2 Hz), in line with literature trends.

Simulation in Abaqus

  • Built a planar FE model to tune geometry, preload, and stiffness characteristics.
  • Modal analysis followed by base-excitation runs produced displacement histories used to compute transmissibility.
  • Simulation trends matched the experiments qualitatively and guided design variants.
